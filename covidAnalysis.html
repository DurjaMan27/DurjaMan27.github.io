Covid-19 Case Analysis and Prediction

Check out our code here!

March 2020 Varun Rao

For 2 years, the Glen Pi Geeks group worked with the Raspberry Pi and various physical hardware components to create complex projects. Now, however, the group became interested in more coding-focused projects, specifically related to data science. After learning the basics of data science with Python through DataCamp's Data Scientist with Python course, we began searching for a topic to analyze. However, at the time, our whole world changed due to the Covid-19 pandemic and quarantine. As a result of the current events surrounding us, we decided to start analyzing Covid-19 cases using the public Johns Hopkins database. We used this database to analyze cases and ultimately experiment with machine learning models to predict how cases may rise/fall in different countries around the world.
